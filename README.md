# Neotec Dual Sync

A settings-driven Frappe app for controlled synchronization between isolated ERPNext/Frappe instances.

## Features
- Select which DocTypes to sync using rules
- Trigger mode: On Submit, Batch, Both, or Manual
- No hardcoded document logic in the core pipeline
- API key/secret + shared secret settings
- Queue-based processing and retry handling
- Sync logs with payload/response capture
- Child-table field mapping rules
- Frappe Cloud-safe package structure
